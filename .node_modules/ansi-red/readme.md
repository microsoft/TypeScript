# ansi-red [![NPM version](https://badge.fury.io/js/ansi-red.svg)](http://badge.fury.io/js/ansi-red)

> The color red, in ansi.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i ansi-red --save
```

## Usage

```js
var red = require('ansi-red');
```
